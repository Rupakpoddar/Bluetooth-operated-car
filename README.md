# Parts required:
- Couple of motors and wheels
- Arduino board
- HC05 Bluetooth module
- DUAL Motor driver
- Jumper wires
- Batteries

## Optional content:
### To add speed control:
- DUAL Motor driver with PWM speed control
### To add camera:
- ESP32 Cam(recommended) or any IP/Wi-Fi camera
